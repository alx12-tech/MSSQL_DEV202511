 # Описание содержимого папки

 ## Назначение

 Папка содержит материалы проекта по курсу MSSQL Developer

 ## Структура

 | SubFolder |Описание|
 |DBStruct  | скрипты БД (создания структуры, объектов, процедур, обслуживания) |
 |Reports   | скрипты отчётов (витрин, представлений)|
 |Pics      | набор изображений для включения в документацию |
 |ТЗ        | Материалы технического задания и для технического задания, презентация (если будет)|
